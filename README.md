# AI-Interviewer
Mini AI Interviewer is a local AI-powered interview simulation tool built with Python and GPT4All that generates topic-based interview questions, conducts interactive sessions, summarizes responses, and stores results — fully offline using a local LLM
This project uses the Orca Mini GPT4All model.

1. Download the model manually from:  
   https://gpt4all.io/models/

2. Place the model in the local folder.
